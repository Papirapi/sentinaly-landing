Fonts used by the landing page (all active via @font-face in styles.css):

1. The Acres Sans  -> TheAcresSans.otf
   (uppercase display: hero headline, section labels, footer eyebrow)

2. Neue Grand 53 Regular Wide -> NeueGrand-Wide-Regular.otf
   (headings, body copy, module list, buttons, footer)

3. Geist -> loaded from Google Fonts
   (nav menu + CTAs, "Play video / 2 min")

To swap a font, replace the .otf file here (keep the same name) or edit the
@font-face src paths at the top of styles.css. .woff2 will load faster than
.otf if you want to convert them (e.g. via https://transfonter.org).
